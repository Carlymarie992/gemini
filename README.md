# Gemini - District Court Case Management Assistant

An AI assistant that helps you manage your district court case against an offender and abuser who has DARVO'd you in court.

## What is DARVO?

DARVO stands for **Deny, Attack, and Reverse Victim and Offender** - a manipulation tactic commonly used by abusers when confronted with their behavior. This assistant helps you:

- Document and recognize DARVO tactics
- Organize your case information and evidence
- Track timeline of events
- Prepare for court appearances
- Maintain detailed records for legal proceedings

## Features

### 1. Case Information Management
- Store essential case details (case number, parties, court, judge)
- Keep all information organized in one place

### 2. Timeline Tracking
- Document events chronologically
- Record dates, locations, and witnesses
- Build a clear narrative of what happened

### 3. Evidence Management
- Catalog all types of evidence (documents, photos, videos, witness statements)
- Track where evidence is stored
- Add detailed notes for each piece of evidence

### 4. DARVO Incident Documentation
- Structured recording of DARVO tactics
- Document denial, attack, and reversal patterns
- Build evidence of manipulation tactics

### 5. DARVO Tactics Guide
- Learn to recognize manipulation tactics
- Understand why documentation matters
- Get tips for court proceedings

### 6. Court Preparation
- Comprehensive checklist for before, during, and after court
- Testimony guidelines
- Tips for staying factual and composed

### 7. Case Export
- Generate complete case summaries
- Create reports for legal review
- Export all documentation to text files

## Installation

### Prerequisites
- Node.js 12.0.0 or higher

### Setup

1. Clone or download this repository:
```bash
git clone https://github.com/Carlymarie992/gemini.git
cd gemini
```

2. Make the script executable (optional, for direct execution):
```bash
chmod +x index.js
```

## Usage

Run the application using Node.js:

```bash
npm start
```

Or directly:
```bash
node index.js
```

### First Time Setup

1. Start the application
2. Select option 2 to "Update Case Information"
3. Enter your case details
4. Begin documenting your case

### Daily Use

- Add timeline events as they occur
- Document DARVO incidents immediately when they happen
- Add evidence as you collect it
- Review the DARVO guide regularly
- Use the court preparation checklist before hearings

## Data Storage

All data is stored locally in a `.case-data` directory in your current working directory:
- `case.json` - All case information
- `case-summary-[timestamp].txt` - Exported summaries

**Important:** This data is stored locally on your computer. Keep backups and ensure your files are secure.

## Menu Options

1. **View Case Information** - Display your case details
2. **Update Case Information** - Add or modify case information
3. **Add Timeline Event** - Record events chronologically
4. **Add Evidence** - Catalog evidence items
5. **Document DARVO Incident** - Record manipulation tactics
6. **View DARVO Tactics Guide** - Learn about DARVO
7. **Add Note** - Add general notes or observations
8. **View All Records** - See complete case documentation
9. **Court Preparation Checklist** - Review preparation steps
10. **Export Case Summary** - Generate a text report
0. **Exit** - Close the application

## Tips for Effective Use

### Documentation Best Practices
- Be specific with dates and times
- Record witness names when possible
- Describe events factually, not emotionally
- Document patterns, not just individual incidents
- Keep evidence organized and accessible

### DARVO Recognition
- Note when facts are denied despite evidence
- Record character attacks or credibility challenges
- Identify when the abuser claims victimhood
- Document the pattern across multiple incidents

### Court Preparation
- Review all documentation before court dates
- Practice staying calm and factual
- Prepare for potential DARVO tactics
- Bring organized copies of all evidence
- Arrive early and dress professionally

## Security and Privacy

- All data is stored locally on your device
- No data is transmitted over the internet
- Keep your computer and files secure
- Create backups of the `.case-data` directory
- Consider encrypting sensitive files
- Consult with your attorney about what to document
